# ♟️ JS Chess Core (Lightweight Engine)

A smart chess board built entirely with **Vanilla JavaScript**, focused on clean, optimized, and highly compressed game logic.

## 🚀 Features
* **Ultra-Lightweight:** The entire core game logic and complex piece movements are compressed into fewer than **210 lines of code**!
* **Optimized Movements:** Full movement support for the Bishop and Rook, smoothly merged using smart functions to dynamically calculate the Queen's path.
* **Auto-Rotating Board:** Features an automatic 180-degree board rotation (`nextPlayer`) after every turn for a seamless local multiplayer experience.
* **Smart Win Detection:** A native tracking system detects the presence of kings, ending the game instantly and dynamically changing the board's background color to match the winner.

## 🛠️ Tech Stack
* HTML5 / CSS3
* JavaScript (ES6+) - DOM Manipulation & Game Logic
